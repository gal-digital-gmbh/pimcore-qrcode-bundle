# Pimcore QR-Code bundle

Adds a backend configuration view for QR-Codes.

## Requirements

Pimcore >= 10.5

## Installation

Require the bundle

```bash
composer require gal-digital-gmbh/pimcore-qrcode-bundle
```

Enable the bundle

```bash
php bin/console pimcore:bundle:enable PimcoreQrcodeBundle
```
